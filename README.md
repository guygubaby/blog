  ```shell script
  git clone https://github.com/guygubaby/blog
  ```

  ```shell script
  yarn dev
  ```

- deploy the blog
  ```shell script
  ./deploy.sh
  ```

- push to GitHub
  ```shell script
  ./commit.sh
  ```
